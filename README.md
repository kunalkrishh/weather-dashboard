# 🌦️ Live Weather Dashboard

A modern and responsive Weather Dashboard built with **Python**, **Streamlit**, and **WeatherAPI**.

## 🚀 Live Demo

👉 https://weather-dashboard-dt3qkjk7hgburudsqkkptu.streamlit.app

---

## 📌 Features

- 🌍 Search weather by city name
- 🌡️ Real-time temperature
- 💧 Humidity
- 💨 Wind Speed
- 🌍 Pressure
- 👀 Visibility
- ☀️ UV Index
- ☁️ Cloud Coverage
- 🌬️ Wind Direction & Gust
- 📍 Location Details
- 📱 Responsive Design

---

## 🛠️ Tech Stack

- Python
- Streamlit
- WeatherAPI
- Requests
- HTML/CSS

---

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/kunalkrishh/weather-dashboard.git
```

Go to the project folder

```bash
cd weather-dashboard
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the app

```bash
streamlit run app.py
```
